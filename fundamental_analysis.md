### **AGENT 4: Fundamental Analysis (Lightweight for Intraday)**

**For intraday/scalping, keep this minimal but important:**

**Real-time Fundamentals:**
- Upcoming earnings date (avoid if within 2 days)
- Institutional holding changes (quarterly)
- Promoter pledge status
- Block deals/bulk deals detected today

**Relative Valuation:**
- P/E vs sector P/E
- EV/EBITDA vs peers
- ROE, ROCE trends

**Corporate Actions:**
- Dividend ex-dates
- Stock split/bonus announcements
- Buyback programs

**Sentiment Indicators:**
- Analyst rating changes
- Target price revisions
- Insider transactions

**Output:** GO/NO-GO signal + risk level
